PixelFrame Photography Studio
A modern, responsive photography studio website with interactive features and vibrant gradient design.
Features

Modern Design - Vibrant gradients and smooth animations
Fully Responsive - Works on all devices
Portfolio Gallery - Filterable image showcase
Cost Calculator - Interactive pricing tool
Contact Form - With file upload support
Fast & Lightweight - No frameworks required

Quick Start

Download all files
Open index.html in your browser
That's it!

Files

index.html - Main structure
style.css - Styling & responsive design
app.js - Interactive functionality

Customization
Colors: Edit CSS variables in style.css
css:root {
    --deep-purple: #6366F1;
    --vibrant-pink: #EC4899;
}
Services/Prices: Update in both Services section and Calculator section
Images: Replace Unsplash URLs with your own
Contact Info: Update address, phone, email in Contact section
Tech Stack

HTML5
CSS3 (Gradients, Flexbox, Grid)
Vanilla JavaScript
Google Fonts (Playfair Display, Inter)
Font Awesome Icons
